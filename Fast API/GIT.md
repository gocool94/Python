# Introduction

# GIT BASICS
-  'git init' - initialises a git repository
-  'git add .'- Adds files from non-stagging area to staging area 
-  'git commit -m "info here' - Move file from staging area to a commit
-  git checkout <commit#> - Open up previous commit
-  'git log' - shows commited log
- 

- git branch branchname - create a branch
- git checkout branchname - goes to branch
- git branch -d new-branch - delete the new branch

- git remote add origin <url>
- git push -u origin main
